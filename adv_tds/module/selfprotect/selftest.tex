\moduledescription{mod.selfprotect.selftest.desc}

The module \tds{mod.selfprotect.selftest} of subsystem \tdslink{sub.selfprotect}
provides functionality to run self-tests of the TSF.

\moduleprocesses{prc.selfprotect.selftest}

\moduleprocess{prc.selfprotect.selftest.runselftest}{Run Self-Tests}

This process runs a self test of the TSF. The process is run at boot-up and
periodically every 24 hours. The process can also be called by an administrator.

\implementedsfr{\sfrlink{fpt_tst.1}}

\moduleinterfaces{int.selfprotect.selftest}

\moduleinterfaceprovided{int.selfprotect.selftest.runtest}

This interface calls the self test (see
\autoref{prc.selfprotect.selftest.runselftest}).
