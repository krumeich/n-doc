\moduleprocesses{prc.vpn.cert}

\moduleprocess{prc.vpn.cert.checkcert}{Verification of the VPN concentrator certificate}

The certificate is checked mathematically and for validity. The expiry date must
be at least one day in the future. The SHA-256 hash of the certificate is
calculated by calling the function
\tdslink[fq]{int.cryptsystem.algorithms.gethash}.

\implementedsfr{\sfrlink{fpt_tdc.1/zert}}

\moduleinterfaces{int.vpn.cert}


\moduleinterfaceprovided{int.vpn.cert.checkcert}

This interface is called to check the certificate of a VPN concentrator (see
\autoref{prc.vpn.cert.checkcert}).

\moduleinterfacerequired{int.cryptsystem.algorithms.gethash}

The interface \tdslink[fq]{int.cryptsystem.algorithms.gethash} is required to
calculate the hash value of the certificate.
