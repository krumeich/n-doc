\moduledescription{mod.cryptsystem.algorithms.desc}

Module \tds{mod.cryptsystem.algorithms} of subsystem \tdslink{sub.cryptsystem}
provides cryptographic base functionalities.


\moduleprocesses{prc.cryptsystem.algorithms}

\moduleprocess{prc.cryptsystem.algorithms.hash}{Calculate Hash Values}

This process calculates SHA-2 hash values.

\implementedsfr{\sfrlink{fcs_cop.1/hash}}

\moduleprocess{prc.cryptsystem.algorithms.hmac}{Calculate HMAC}

This process calculates HMAC.

\implementedsfr{\sfrlink{fcs_cop.1/hmac}}

\moduleinterfaces{int.cryptsystem.algorithms}

\moduleinterfaceprovided{int.cryptsystem.algorithms.gethash}

This interface triggers the hash value calculation (see
\autoref{prc.cryptsystem.algorithms.hash})

\moduleinterfaceprovided{int.cryptsystem.algorithms.gethmac}

This interface triggers the HMAC calculation (see
\autoref{prc.cryptsystem.algorithms.hmac})
