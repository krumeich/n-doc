\hrefsubsection{intro.desc.arch}{TOE Architecture and Functionalities}

The TOE consists of the following subsystems:

\begin{description}
\item[\tds{sub.vpn}] contains VPN functionalities such as enthält die
  VPN-Funktionen IPSec and IKE.
\item[\tds{sub.ntpclient}] synchronizes system time with an NTP server in the WAN.
\item[\tds{sub.selfprotect}] contains protection mechanisms for the TOE.
\item[\tds{sub.adminsystem}] provides the management functions of the TOE.
\item[\tds{sub.cryptsystem}] provides cryptographic functionalities.
\item[\tds{sub.tls}] creates TLS connections for the management console.
\end{description}
